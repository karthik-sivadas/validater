    2. **Replace img with canvas** (`packages/web/src/components/live-viewer.tsx`):
       Replace the `<img>` element in the frame display section with a canvas-based renderer:
       - Add `useRef<HTMLCanvasElement>(null)` for the canvas element.
       - Create a `BrowserCanvas` component (or inline in LiveViewer) that:
         - Takes `frame: string | null` as a prop.
         - Uses `useEffect` to draw frames: create an offscreen `Image` object, set `onload` to draw to canvas via `ctx.drawImage()`, set `src` to `data:image/jpeg;base64,${frame}`.
         - Only update canvas dimensions when they differ from the image (avoids canvas clear on every frame).
         - The canvas element should have `className="w-full h-full"` and the container should keep `className="aspect-video overflow-hidden rounded-lg border"`.
       - The "Waiting for stream..." placeholder stays as-is (shown when no frame yet).

    3. **Add action badges to StepRow** (`packages/web/src/components/live-viewer.tsx`):
       Update the `StepRow` component:
       - Add a color map for action types:
         ```
         const ACTION_COLORS: Record<string, string> = {
           click: "bg-blue-100 text-blue-800 dark:bg-blue-900 dark:text-blue-200",
           fill: "bg-purple-100 text-purple-800 dark:bg-purple-900 dark:text-purple-200",
           assert: "bg-amber-100 text-amber-800 dark:bg-amber-900 dark:text-amber-200",
           navigate: "bg-green-100 text-green-800 dark:bg-green-900 dark:text-green-200",

           hover: "bg-cyan-100 text-cyan-800 dark:bg-cyan-900 dark:text-cyan-200",
           wait: "bg-gray-100 text-gray-800 dark:bg-gray-900 dark:text-gray-200",
         };
         ```
       - In StepRow, add the action badge between the pass/fail badge and the duration:
         ```
         <span className={`inline-flex items-center rounded-md px-1.5 py-0.5 text-xs font-medium ${ACTION_COLORS[step.action] ?? ACTION_COLORS.wait}`}>
           {step.action}
         </span>
         ```
       - Replace the "Step {step.stepOrder}" text with the action badge.
       - Add a description line below the badge row: `<p className="text-xs text-muted-foreground truncate">{step.description}</p>`
       - Keep the error display as-is.
