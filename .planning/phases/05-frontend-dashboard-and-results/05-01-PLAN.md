    **Create `packages/web/src/server/test-runs.ts`** with two server functions:

    1. **getTestRunList** - GET, accepts `{ page, pageSize, status?, search? }`:
       - Use `createServerFn({ method: "GET" })` with `.inputValidator()` accepting a Zod schema
       - Dynamic import `@validater/db` (db, testRuns), `drizzle-orm` (eq, desc, like, and, sql, count), `@tanstack/react-start/server` (getRequestHeaders), and `@/lib/auth` (auth)
       - Get authenticated user via `getRequestHeaders()` + `auth.api.getSession({ headers })`. Throw Error("Unauthorized") if no session.
       - Build where conditions: always filter by `eq(testRuns.userId, session.user.id)`. Optionally add status filter (skip if "all" or undefined) and search filter (like on url).
       - Get total count using `db.select({ total: count() }).from(testRuns).where(whereClause)`. If `count` is not importable from drizzle-orm, use `sql<number>\`count(*)\`` as fallback.
       - Get paginated runs using `db.select().from(testRuns).where(whereClause).orderBy(desc(testRuns.createdAt)).limit(pageSize).offset((page-1)*pageSize)`
       - Return `{ runs, total, page, pageSize, totalPages: Math.ceil(total/pageSize) }`

    2. **getTestRunDetail** - GET, accepts `{ testRunId: string }`:
       - Dynamic import `@validater/db` (db, testRuns, testRunResults, testRunSteps), `drizzle-orm` (eq, asc)
       - Query test run by id. Return null if not found.
       - Query testRunResults where testRunId matches.
       - For each result, query testRunSteps where resultId matches, ordered by `testRunSteps.stepOrder` ascending.
       - Return `{ run, results: resultsWithSteps }` where each result includes its steps array.

    **IMPORTANT patterns from existing codebase:**
    - Use dynamic `import()` for ALL server-only deps inside the handler (same pattern as run-test.ts)
    - Use `db.select().from().where()` builder API, NOT relational query API
    - Import `count` from `drizzle-orm` -- if TypeScript errors, use `sql<number>\`count(*)\`` instead

      ```typescript
      const session = await auth.api.getSession({ headers });
      if (!session) throw new Error("Unauthorized");
      ```
    - Change `userId: "anonymous"` to `userId: session.user.id` in the db.insert call
