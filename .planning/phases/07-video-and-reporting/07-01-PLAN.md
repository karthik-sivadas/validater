   - Before creating BrowserContext, create a temp directory:

     ```typescript
       dir: tempVideoDir,

     },
     ```
   - After step execution completes but BEFORE `context.close()`, capture the video reference:
     ```typescript
     const pages = context.pages();
     const videoRef = pages[0]?.video();
     ```
   - The existing `context.close()` in the `finally` block triggers video file finalization.
   - After `context.close()` in the finally block, save the video (best-effort, wrapped in try/catch):
     ```typescript
     let videoRelativePath: string | undefined;
     if (videoRef) {
       try {
         const videoFilePath = await videoRef.path();
         videoRelativePath = await saveVideo(
           params.streamingConfig?.testRunId ?? '',
           params.viewport.name,
           videoFilePath,
         );
       } catch {
         // Video save failure must not break execution
       }
     }
     // Cleanup temp directory (best-effort)
     await rm(tempVideoDir, { recursive: true, force: true }).catch(() => {});
     ```
   - **IMPORTANT**: The `videoRelativePath` must be accessible in the return value. Restructure the try/finally so that `videoRelativePath` is defined at the outer scope and set inside the finally. The return statement must include `videoPath: videoRelativePath`.
   - The return object should add `videoPath: videoRelativePath` to the ExecutionResult.
