2. **Create HTML report generator** at `packages/worker/src/reports/html-generator.ts`:
   - Export `generateHtmlReport(data: ReportData): string`.
   - Define `ReportData` interface:
     ```typescript
     interface ReportStep {
       stepOrder: number;
       action: string | null;
       description: string | null;
       status: string;
       durationMs: number;
       errorMessage: string | null;
       errorExpected: string | null;
       errorActual: string | null;
       screenshotBase64: string | null;
     }
     interface ReportViewport {
       viewport: string;
       totalDurationMs: number;
       steps: ReportStep[];
     }
     interface ReportData {
       testRunId: string;
       url: string;
       testDescription: string;
       createdAt: string;
       completedAt: string | null;
       status: string;
       viewportResults: ReportViewport[];
     }
     ```
   - Read the HTML template from disk using `fs/promises` with `import.meta.url` for path resolution: `new URL('./templates/report.html', import.meta.url)`.
   - Replace `{{testRunId}}`, `{{url}}`, `{{testDescription}}`, `{{createdAt}}`, `{{status}}` placeholders.
   - Compute summary stats (totalSteps, passed, failed, passRate) and replace those placeholders.
   - Generate viewport sections HTML by iterating `viewportResults` and building step card HTML strings. Replace `{{VIEWPORT_SECTIONS}}` with the concatenated HTML.
   - For screenshots, embed as `<img src="data:image/png;base64,{base64}" />` directly in the HTML.
   - Return the complete HTML string.

   - Close browser after PDF generation.
   - Return the PDF buffer.
   - Wrap in try/finally to ensure browser is always closed.

   - **`exportPdfReport`**: Server function (GET) accepting `{ testRunId: string }`.
     - Same auth and data fetch as `exportHtmlReport`.
     - Call `generateHtmlReport(reportData)` to get HTML.
     - Dynamically import `generatePdfReport` from `@validater/worker`.
     - Call `generatePdfReport(html)` to get PDF buffer.
     - Convert buffer to base64: `pdfBuffer.toString('base64')`.
     - Return `{ pdfBase64: base64String, filename: 'validater-report-{testRunId}.pdf' }`.

2. **Add export buttons to the results page** in `packages/web/src/routes/_authed/runs/$runId.tsx`:
   - Import `exportHtmlReport` and `exportPdfReport` from `@/server/exports`.
   - Add a `[useState]` for `exporting` state (null | 'html' | 'pdf') to show loading state.
   - Add an export actions row in the header area (between the URL/description and the Summary card):
     ```tsx
     <div className="flex items-center gap-2">
       <Button
         variant="outline"
         size="sm"
         disabled={exporting !== null}
         onClick={async () => {
           setExporting('html');
           try {
             const result = await exportHtmlReport({ data: { testRunId: run.id } });
             // Trigger download via Blob
             const blob = new Blob([result.html], { type: 'text/html' });
             const url = URL.createObjectURL(blob);
             const a = document.createElement('a');
             a.href = url;
             a.download = result.filename;
             a.click();
             URL.revokeObjectURL(url);
           } finally {
             setExporting(null);
           }
         }}
       >
         {exporting === 'html' ? 'Exporting...' : 'Export HTML'}
       </Button>
       <Button
         variant="outline"
         size="sm"
         disabled={exporting !== null}
         onClick={async () => {
           setExporting('pdf');
           try {
             const result = await exportPdfReport({ data: { testRunId: run.id } });
             // Decode base64 and trigger download
             const binary = atob(result.pdfBase64);
             const bytes = new Uint8Array(binary.length);
             for (let i = 0; i < binary.length; i++) bytes[i] = binary.charCodeAt(i);
             const blob = new Blob([bytes], { type: 'application/pdf' });
             const url = URL.createObjectURL(blob);
             const a = document.createElement('a');
             a.href = url;
             a.download = result.filename;
             a.click();
             URL.revokeObjectURL(url);
           } finally {
             setExporting(null);
           }
         }}
       >
         {exporting === 'pdf' ? 'Exporting...' : 'Export PDF'}
       </Button>
     </div>
     ```
   - Only show export buttons when `run.status === 'complete'` (no exports for in-progress or failed runs).
